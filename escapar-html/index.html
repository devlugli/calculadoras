HTML escapado

Para que serve escapar HTML?

Essa ferramenta é útil quando você precisa mostrar código HTML em páginas, materiais de apoio, documentação e tutoriais sem que o navegador interprete as tags.

O que é convertido?

Caracteres especiais como <, >, aspas e & são transformados em entidades HTML seguras.

Quando usar essa ferramenta?

Use o escape de HTML ao publicar exemplos de código, snippets, documentação técnica, blocos de tutorial e conteúdos educacionais.
