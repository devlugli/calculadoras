O que são juros simples?

Juros simples são calculados apenas sobre o valor inicial do capital. Diferente dos juros compostos, os juros não se acumulam sobre os juros dos períodos anteriores.

Fórmula dos juros simples

A fórmula utilizada é:

J = C × i × t

Onde:

  • C = capital inicial
  • i = taxa de juros
  • t = tempo

Onde os juros simples são usados?

Esse cálculo é comum em empréstimos de curto prazo, cálculos escolares, negociações comerciais e algumas operações financeiras simples.

Perguntas frequentes

Qual a diferença entre juros simples e compostos?

Nos juros simples o cálculo é sempre feito sobre o capital inicial. Nos juros compostos os juros acumulam sobre o valor atualizado.

Quando usar juros simples?

Esse modelo é usado quando os juros não são capitalizados ao longo do tempo.
