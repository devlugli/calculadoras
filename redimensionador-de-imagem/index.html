Perguntas frequentes

A ferramenta envia meus dados para servidor?

Não. O processamento acontece no navegador e não exige backend, login ou API externa.

Funciona no celular?

Sim. A interface usa o layout responsivo do CalcBrazil e os botões foram pensados para toque.

Posso usar de graça?

Sim. A ferramenta é gratuita e funciona diretamente na página.

O resultado é definitivo?

Use o resultado como base prática e revise antes de publicar, enviar ou salvar.

Preciso instalar algo?

Não. Basta abrir a página no navegador.
