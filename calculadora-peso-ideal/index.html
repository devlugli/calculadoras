Calculadora de peso ideal online

Nossa calculadora de peso ideal ajuda você a descobrir uma estimativa de peso saudável com base no sexo e na altura informados. O cálculo usa um IMC de referência para gerar um valor aproximado.

Como calcular o peso ideal?

Nesta calculadora, o peso ideal é estimado com base em um IMC de referência. Usamos IMC 22 para homens e IMC 21 para mulheres.

Fórmula usada

Peso ideal = IMC de referência × altura²

O que é considerado peso ideal?

O chamado peso ideal é apenas uma estimativa teórica usada como referência. O peso saudável real pode variar conforme idade, composição corporal, massa muscular e outros fatores.

Para que serve essa calculadora?

Essa calculadora serve como apoio rápido para quem quer ter uma noção inicial de qual seria um peso aproximado para sua altura.

Perguntas frequentes sobre peso ideal

Como é calculado o peso ideal?

O cálculo é feito multiplicando um IMC de referência pela altura ao quadrado.

Existe diferença entre homem e mulher?

Sim. Nesta calculadora usamos uma referência diferente para homens e mulheres.

Esse resultado é exato?

Não. O resultado é apenas uma estimativa de referência.

Essa calculadora substitui orientação médica?

Não. Ela serve apenas como referência inicial.
