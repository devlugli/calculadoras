Como funciona a calculadora?

A calculadora aplica a fórmula de Bhaskara para resolver expressões do tipo ax² + bx + c = 0 e identificar as raízes reais quando elas existem.

O que significa o valor de delta?

Quando delta é positivo, existem duas raízes reais diferentes. Se for zero, existe uma raiz real dupla. Se for negativo, não há raízes reais.
